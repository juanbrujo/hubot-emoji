# Hubot Emoji

**Hubot emoji find & replace from a given string**

### Installation:

````
$ npm install hubot-emoji --save
````

Add in `external-scripts.json`:

````
[
  ..
  "hubot-emoji"
]
````

### Use:

````
hubot> hubot emoji <string>
````
